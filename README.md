### SelectionSort & InsertionSort

#### Compilando

**`$ make `**

#### Executando

Para testar as instancias basta executar usando o parametro <"arquivo de entrada".in> seguido do nome do arquivo de saida "resultado".out .

##### Exemplo de execução

**`$ ./selectionSort <num.1000.1.in> output.out`**

**`$ ./insertionSort <num.1000.1.in> output.out`**
